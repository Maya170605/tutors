# tutors

Веб-приложение для автоматизации работы репетиторов.

##  Описание

Tutors Management System - это веб-приложение для управления базой данных репетиторов. Система предоставляет возможности просмотра, добавления, редактирования и удаления информации о репетиторах, а также управление пользователями с различными уровнями доступа.

##  Функциональности

###  Управление пользователями
- **Регистрация и аутентификация** пользователей
- **Ролевая модель**: Администратор и Пользователь
- **Разграничение прав доступа** к функционалу

###  Управление репетиторами
- **Просмотр списка** всех репетиторов
- **Поиск и фильтрация** по предметам и опыту
- **Добавление новых** репетиторов (только для админов)
- **Редактирование информации** о существующих репетиторах
- **Удаление репетиторов** из базы данных

###  Информация о репетиторах
- Полное имя репетитора
- Образование и квалификация
- Опыт работы
- Преподаваемые предметы
- Фотография репетитора
- Дата добавления в систему

### База данных
- **MySQL 8.0** - система управления базами данных
